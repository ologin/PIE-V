How to Correctly Make Mistakes: A Framework for Constructing and Benchmarking Mistake Aware Egocentric Procedural Videos

Olga Loginova and Frank Keller

Paper Code Release

Abstract

PIE-V builds controlled mistake-aware procedural video examples from clean egocentric procedures. It samples cognitively plausible mistakes, models corrections, rewrites procedures with an LLM, validates the resulting text, and links the plan to video editing case studies.

Error taxonomy

Five controllable procedural mistake types

Wrong execution A step is performed with an incorrect argument, object, location, or manner.
Deletion A required procedural step is skipped.
Substitution A step is replaced by a plausible but incorrect alternative.
Insertion An unnecessary step is inserted into the procedure.
Transposition Two steps are performed in an incorrect order.

Repository

Code, generated text plans, and aggregate annotations

The repository includes semantic-role resources, sampled PIE-V plans, judged instruction rewrites, a 50-take name list, and an aggregate annotation summary. Ego-Exo4D source videos and original annotation files are local inputs governed by the Ego-Exo4D access terms.
