# Example Outputs

This directory contains compact examples from the text side of the PIE-V pipeline:

- `split_50_error_plan.json`: sampled structured error plan.
- `split_50_error_plan_with_corrections.json`: the same plan after correction simulation.
- `split_50_error_instructions_openai.json`, `split_50_error_instructions_qwen.json`: writer outputs.
- `split_50_error_instructions_openai_judged.json`, `split_50_error_instructions_qwen_judged.json`: judged outputs.
- `openai_judged_report.csv`: example judge report.
- `split_50_llm_freeform_errors_openai.json`, `split_50_llm_freeform_errors_qwen.json`: freeform LLM baselines.
- `openai_diagnostics.csv`, `qwen_diagnostics.csv`: baseline diagnostics.
- `new_annotations/`: updated split-style annotations for several video edit case studies.
