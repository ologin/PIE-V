# Data Directory

This directory contains lightweight, publication-oriented artifacts used by PIE-V.

- `resources/`: semantic roles, predicate-role priors, vocabulary, and cached semantic representations.
- `manifests/`: take-name list for the 50-scenario Ego-Exo4D subset.
- `examples/`: generated PIE-V plans, judged instruction rewrites, readable logs, and example updated annotations.
- `annotations/`: aggregate human annotation summary exported from the `DATA` sheet of `Annotation_Summary.xlsx`.

Raw Ego-Exo4D videos, original Ego-Exo4D annotation files, generated clips, and
provider-specific working directories are local inputs or release media.
