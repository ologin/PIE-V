# Resources

Small semantic resources used by the planner and validator:

- `split_50_vocabulary.csv`: step text to vocabulary id map for the 50-scenario subset.
- `semantic_representations_split_50.json`: predicate-role semantic representations for the subset.
- `semantic_roles.csv`: role impact assignments.
- `semantic_roles_by_predicate.csv`: predicate-conditioned role priors.
- `role_predicate_pairs.csv`, `structural_roles.csv`, `embedded_predicates.csv`: helper summaries for semantic-role analysis.
